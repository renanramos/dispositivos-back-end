Dispositivos back-end

GET

http://localhost:5000/devices : consulta todos os dispositivos cadastrados.

http://localhost:5000/devices/:deviceId : consulta dispositivo por id.

http://localhost:5000/devices/search?modelo= : consulta dispositvo por modelo

POST

http://localhost:5000/devices realiza a inclusão de um novo dispositivo ao sistema.

PUT

http://localhost:5000/devices realiza a alteração do dispositivo

DELETE

http://localhost:5000/devices/:deviceId : deleta o dispositivo da base de dados
